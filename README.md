# Simple API Server

## Packages

```
npm i --save \
mysql \
knex \
bookshelf \
bcrypt \
body-parser \
cors \
express

```

```
npm i --save-dev \
babel-plugin-add-module-exports \
babel-preset-es2015 \
babel-preset-react \
babel-preset-stage-2 \
babel-register
```
## npm scripts

```
$ npm run make:migrate users_create
$ npm run make:seed users
$ npm run migrate
$ npm run rollback
$ npm run seed

```

## run server

```
$ npm start
```
